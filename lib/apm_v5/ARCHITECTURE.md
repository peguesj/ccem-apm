# CCEM APM v5 — Module Architecture Taxonomy

Generated by Wave 1 formation agent `sq1-arch-4` as part of v8.0.0 modularization.

## Categories

- **Foundation** — always-on infrastructure; must never be removed or extracted
- **Plugin-candidate** — a self-contained APM feature that wraps one or more GenServers; suitable for extraction into `lib/apm_v5/plugins/<name>/`
- **Integration-candidate** — a bidirectional bridge to an external protocol; suitable for extraction into `lib/apm_v5/integrations/<name>/`
