## US-035 - Refactor dependency graph for collapsible hierarchy with curved connectors
- Complete rewrite of dependency_graph.js from force-directed (665 lines) to d3.tree() hierarchy (568 lines)
- Collapsible nodes via _toggleChildren() swapping d.children and d._children
- d3.linkVertical() for curved Bezier connector paths instead of straight lines
- SVG arrowhead markers in 4 color variants (default, active, error, warning)
- 750ms transitions with d3.easeCubicInOut for smooth expand/collapse
- Handles both hierarchy_data (new) and agents_updated (backward compat) events
- Keyboard navigation: Enter/Space expand/collapse, Arrow keys navigate tree
- Expand state preservation across data refreshes
- Files changed: assets/js/hooks/dependency_graph.js (REWRITE)
- Learnings: D3 enter/update/exit pattern with position snapshots (x0/y0) is key for smooth hierarchical transitions
